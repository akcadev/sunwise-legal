# sunwise-legal

Public legal pages for the SunWise iOS app, served via GitHub Pages.

- `privacy.html` — Privacy Policy
- `terms.html` — Terms of Use (EULA)
- `index.html` — landing page linking both

This repo is intentionally separate from the private app repo so that **only** these
pages are ever public. Source code and project docs stay private.

## URLs (after enabling Pages)

- https://akcadev.github.io/sunwise-legal/privacy.html
- https://akcadev.github.io/sunwise-legal/terms.html

## Enabling Pages

Settings → Pages → Source: **Deploy from a branch** → Branch: `main` / folder: `/ (root)`.
